# Mermaid Diagrams Test

Testing different types of Mermaid diagrams using the standard markdown syntax.

## Flowchart

```mermaid
flowchart TD
    A[Start] --> B{Is it working?}
    B -->|Yes| C[Great!]
    B -->|No| D[Debug it]
    D --> E[Fix the issue]
    E --> B
    C --> F[Deploy]
    F --> G[End]
```

## Sequence Diagram

```mermaid
sequenceDiagram
    participant Client
    participant API
    participant Database
    
    Client->>API: POST /user
    API->>Database: Save user data
    Database-->>API: User saved
    API-->>Client: 201 Created
```

## Simple Graph

```mermaid
graph LR
    A[Write Code] --> B[Commit]
    B --> C[Push]
    C --> D[Deploy]
```

This approach uses the rehype-mermaid plugin to render diagrams at build time.
